# Projeto: Análise Exploratória de Dados - Netflix Stock

## 📖 Sobre o meu aprendizado
Este projeto faz parte da minha jornada de transição para a área de **Inteligência de Dados**. Como estudante de Engenharia de Software, meu foco aqui é aprender a tratar dados e extrair insights, utilizando IA como mentor técnico.

## 🚀 Minha Evolução
* **Fase Atual:** Limpeza de dados e tratamento de séries temporais.

* **Apoio da IA:** Utilizei o Gemini para me orientar na lógica de programação e para me explicar o comportamento de funções da biblioteca Pandas.

## 🛠 Tecnologias

- Análise: Pandas
- Mentoria Técnica: Google Gemini
